# 🏥 Health Insurance Cost Prediction using Linear Regression

## Overview
This project analyzes a health insurance dataset to understand the factors that influence medical insurance charges. 

Using exploratory data analysis and a linear regression model, the notebook predicts insurance costs based on demographic and lifestyle features such as age, BMI, smoking status, and region.

The goal is to demonstrate a basic machine learning workflow from data exploration to predictive modeling.

---

## Problem Statement
Insurance companies need to estimate medical costs based on customer characteristics.  
This project builds a predictive model to estimate insurance charges using historical data.

---

## Dataset Features
Typical features included in the dataset:

- Age
- Gender
- BMI (Body Mass Index)
- Number of children
- Smoking status
- Region
- Medical insurance charges (target variable)

---

## Project Workflow

1. Data inspection and preprocessing  
2. Exploratory Data Analysis (EDA)  
3. Feature relationship analysis  
4. Data preparation for modeling  
5. Linear Regression model training  
6. Model evaluation and interpretation  

---

## Key Insights

- Smoking status has a significant impact on insurance charges
- Higher BMI tends to increase medical costs
- Age positively correlates with insurance expenses
- Certain demographic patterns influence insurance pricing
