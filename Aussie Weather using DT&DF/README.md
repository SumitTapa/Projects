# 🌦 Rainfall Prediction using Decision Trees & Random Forest

## Overview
This project builds a machine learning classification model to predict whether it will rain the next day using historical weather data from Australia.

The notebook applies Decision Tree and Random Forest algorithms to classify rainfall outcomes based on meteorological features.

---

## Problem Statement
Accurate rainfall prediction is important for agriculture, logistics, and planning.  
This project aims to predict rain occurrence using historical weather observations.

---

## Dataset Features
The dataset includes weather-related attributes such as:

- Temperature (MinTemp, MaxTemp)
- Rainfall
- Humidity
- Wind speed and direction
- Pressure
- Cloud cover
- RainToday (input)
- RainTomorrow (target)

---

## Project Workflow

1. Data cleaning and preprocessing  
2. Handling missing values  
3. Encoding categorical variables  
4. Exploratory Data Analysis (EDA)  
5. Feature selection  
6. Model training:
   - Decision Tree Classifier
   - Random Forest Classifier  
7. Model evaluation:
   - Accuracy
   - Confusion Matrix
   - Feature Importance  

---

## Key Insights

- Humidity and rainfall-related features strongly influence predictions  
- Random Forest performs better than Decision Tree due to ensemble learning  
- Certain weather patterns significantly increase probability of rain
